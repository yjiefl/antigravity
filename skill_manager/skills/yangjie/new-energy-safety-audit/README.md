# 新能源安全生产调度会材料审核专家 - 使用说明

本技能通过模拟资深电力专家，对新能源生产调度会材料进行深度分析。以下是使用步骤和说明：

## 1. 使用方式
您可以采用以下任一方式提供需要审核的材料：
*   **直接粘贴文字**: 将 Word、Excel 或邮件中的文字内容直接复制并粘贴到对话框中。
*   **上传文件**: 直接将材料文件（PDF、Word、Excel 等）发送给 AI，并附上指令“请审核这份材料”。
*   **指定路径**: 如果文件已在工作区，可以直接告知文件路径。

## 2. 操作步骤
1.  **激活**: 在对话输入框中通过特定方式或直接描述需求（如“对以下调度会材料进行审核”）来启动本 Skill。
2.  **输入**: 提交您的材料内容。
3.  **分析**: AI 将调取后台的 `references`（场站容量、计算规则等）进行：
    *   **数据核对**: 校验合计项、百分比、区域数据是否准确（1% 误差红线）。
    *   **逻辑校验**: 检查前后文矛盾、设备容量不匹配等问题。
    *   **管理诊断**: 识别整改措施是否敷衍、是否有具体责任人和时间节点。
4.  **结果**: AI 将输出一份《异常发现清单》和《总结报告》，指出存在的问题及改进建议。

## 3. 注意事项
*   **数据准则**: 审核将严格参考内部的 `stations.md` 场站表，如需更新场站容量，请先修改该参考文件。
*   **多维度审核**: 本 Skill 不仅看数字，还会重点审核整改措施的闭环完整性及因果逻辑。
*   **输出格式**: 默认输出 Markdown 表格，您可以要求输出为 Word 兼容格式或其他。
