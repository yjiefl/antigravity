# 指标异常预警门槛 (Performance Threshold Benchmarks)

本文件定义了各关键指标的“正常运行区间”。当报表数据偏离以下区间时，AI 专家应识别为【性能异常】并要求解释原因，而非仅仅校验计算正确性。

## 1. 效率类指标 (Efficiency)
*   **光伏发电效率**: 
    *   **横向对标**: 场站发电效率偏离同地区平均值 ±15% 以上。
    *   **绝对值预警**: 辐照度 >10MJ/㎡ 时，发电效率低于 60%（除非有限电或大规模故障记录）。
*   **风电等效小时数**: 
    *   **风资源匹配**: 平均风速 >8m/s 时，日等效小时数低于 1.5h（除非有限电）。
*   **储能充放电效率**: 
    *   **系统效率**: 日放电量 / 日充电量 < 75% (需结合 SOC 变化综合判断)。

## 2. 厂用电类指标 (Auxiliary Power)
*   **光伏场站**: 正常区间 [1.5%, 5.0%]。
    *   **黄线预警**: > 8.0%。
    *   **红线预警**: > 15.0% 或出现“负值”。
*   **风电场站**: 正常区间 [1.0%, 3.5%]。
    *   **异常判定**: 极低风速日厂用电率极高属正常，但正常发电日 > 6.0% 需预警。

## 3. 功率预测类指标 (Forecasting)
*   **准确率底线**:
    *   光伏短期准确率 < 70% 或 超短期 < 75%。
    *   风电短期准确率 < 65% 或 超短期 < 70%。
*   **考核电量预警**: 单站月累计考核电量占公司总考核 20% 以上，定为“重点整改对象”。

## 4. 气象参数异常 (Meteorological)
*   **辐照度逻辑**: 夜间 (20:00 - 05:00) 出现辐照度 > 0.1MJ/㎡。
*   **风速逻辑**: 平均风速 > 25m/s（切出风速）但未见损失电量或安全说明。

## 5. 组串故障存量预警
*   **存量红线**: 单站未消缺组串故障数 > 200 条，或存量占总组串 5% 以上。
*   **处理效率**: 新增故障在 48 小时内的消缺率 < 30%。

---

**注**：AI 专家在输出审核报告时，应对触发上述门槛的指标进行【指标异常】标注，并追问其背后的运维原因。
