**4.1** **测量方案**4.1.3 激光雷达、声雷达、超声波测风仪等新型观测设备已逐步应用于风能资源测量、风电场运行期验证试验及科学研究等领域，特别是可移动的激光雷达和声雷达测风仪具有灵活机动的特点，能够有效降低复杂风况区域的风能资源测量成本和评估风险，已初步展现出了较好的工程应用价值。但同时也应当看到，这些新型测风仪器采用与机械式测风设备完全不同的技术原理，相关技术应用仍处于起步阶段，可靠性和准确性仍待大量工程实践的验证，单独使用可能存在一定风险，配合机械式测风设备使用更加稳妥。**4.1.5**观测区域指测风塔的有效覆盖区域。**4.1.6**本规范中测风塔与障碍物的距离规定采用现行国家标准 《风电场风能资源测量方法》GB/T 18709的规定。

4.1.7～4.1.8 简单地形和复杂地形均是对风电场场址地形条件的相对描述，简单地形通常指高程变化较小、起伏不大的场址地形，典型代表如戈壁滩、沙漠、平原、滩涂、草原等，定量判定标准建议为场址区及周边5km 范围内地形总体坡度不超过2°, 局部高差不超过20m, 总体坡度的计算方法是连续场址范围内的总体高程差除以所在区域最大水平距离；复杂地形指高程差异较大、坡度陡峭的场址地形，典型代表如丘陵、山地等，定量判定标准建议为场址区局部高差大于20m。**4.1.9**过渡地形指场址区为简单地形，场址区周边5km 范围内存在复杂地形的地形。

风电场风能资源可能受到周边大范围地形特征或区域气候特**NB/T 31147---2018**征的影响，出现与局部地形特点不相符合的变化规律，例如：在垭口下风向可能存在与局部地形变化不一致的风速变化；在高大山脉周边也可能存在因气候原因造成的风速变化；在沿海区域可能存在与海岸线距离不同造成的风速变化。

4.1.10 安装两层气温计的目的是为了确定测风塔所在位置的大气稳定度，为避免地面热辐射对大气稳定度测量产生不利影响，建议低层气温计安装高度不低于20m, 在初拟风电机组轮毂高度附近安装另一套气温计。

4.1.11 随着风电机组叶轮直径增加，叶轮扫掠面上缘与下缘的风况差异可能扩大，轮毂高度处风况对整个叶轮扫掠面内风况的代表性可能下降，特别是在山地、峡谷、近海、森林等地形或下垫面变化复杂的区域，叶轮扫掠面内的风况可能存在突变，为降低风能资源开发风险，建议加强对整个风轮扫掠面内的风能资源测量，在轮毂高度至扫掠面上缘之间增加测风设备。**4.1.13**特殊要求主要包括对测风塔塔体及测量设备采取的防凝冻、抗台风、防连阴雨等的相应措施。**4.2** **测量数据**

**4.2.1～4.2.3**根据需要也有将记录时间间隔调整为1s 、1min 或 5min 等，建议不大于10min。**4.3** **测量仪器**4.3.2 根据中国气象局编制的《地面气象观测规范》3.2节地面气象观测仪器的基本技术性能要求，风速观测需满足的准确度为0.3$m/s$ +0.03V, 式中V 为观测高度风速。**4.4** **测量设备设施安装**

**4.4.2**通常情况下，为减小塔影效应，同一高度安装2套风速传感器时，安装夹角可选择90°。

在测风塔安装前，可通过现场查勘、中尺度拟合、参考气象站或周边测风塔观测成果等方式，对拟开发区域的主风向和主风能方向进行初步确定，为测风设备安装方向选择提供依据。**4.7** **测量档案**现场照片有助于了解测风塔周边环境情况，照片一般包括主导风向以及东、西、南、北四个方向的环拍照。**5** **风能资源评估**

**5.1** **基础资料**5.1.3 中尺度资料来源可通过应用范围广、认可度高的网络数据库下载，目前行业内采用较多、获取较为便利的再分析数据有 MERRA 、ERA 、NNRP 等；也可通过行业内普遍认可的气象模式计算得到，如 WRF 、MM5 等。**5.2** **测风数据的检验和处理**5.2.1 风速测量不确定性的来源主要有测风设备安装质量和测风仪器性能两个方面，其中测风塔安装质量可从测风塔安装垂直度、水平支臂长度、垂直支杆长度及垂直度、测风仪器安装是否符合规范等角度进行评估。

5.2.5 在山顶、峡谷等特殊地形及气候条件下，近地面风廓线可能出现低层风速高于高层风速的情况，因此，测风数据中反映的负切变是否真实合理需要进行分析论证予以确认，在数据检验时作为异常项是有必要的。

对于风向检验，目前多数测风均仅在10m 高度和测风塔最大高度2处设置有风向传感器，考虑到低层风向受地形及地表附着物影响较大，高低层风向分布特征存在较大差异是可能的，在工程应用时需要进一步分析确定其合理性，本规范建议在风电机组叶轮扫掠面内增加安装一套风向传感器，降低风向测量的不确定性 。

5.2.6 凝冻轻微时可能影响风速传感器测量数据的准确性，凝冻严重时将造成风速、风向观测停止甚至测风塔倒塌，因此在历史发生过凝冻的区域进行风能资源评估时，测风数据检验要特别**NB/T 31147---2018**关注凝冻问题。凝冻能够通过趋势检验发现，当风速或风向数据出现6h 以上的无变化记录，且气温低于5℃、湿度大于80%时，发生严重凝冻的概率较大，且发生严重凝冻的临近时段可能存在风速观测数据不准确的问题。

对于凝冻数据的处理目前有两种较为常用的方法：一种方法是利用同期中尺度模拟数据、激光雷达及声雷达观测数据或未发生凝冻的有效观测数据进行修正，这种方法的优点是处理结果更接近真实情况，评估结果可靠性较高，缺点是可能由于参考数据准确性不足或修正方法局限性带入新的误差，且多数情况下难以获得相关性较高的有效参考数据；另一种方法是直接删除，这种方法的优点是简便易行，处理效率高，缺点是评估结果与真实风速有偏差。

5.2.14 风速数据修正时一般先进行总体相关分析，总体相关系数大于等于0.95时，根据总体相关方程进行修正；总体相关系数小于0.95但大于0.9时，根据总体相关方程或分扇区相关方程进行修正；总体相关系数小于0.9时，进行分扇区相关分析；主风向及主风能方向相关系数均大于0.7且部分扇区大于0.8 时，根据分扇区相关方程进行修正。
