# Chart Studio - 专业统计图表生成器 v1.1.2

一个基于 Web 的高级图形生成器，专注于统计图表的高质量产出，支持自动化持久化与精美视觉效果。

## 🌟 核心特性

- 🥧 **饼图/环形图** - 支持旋转、内部图例、气泡标签、百分比双显。
- 📊 **柱状图** - 支持横向/纵向切换、堆叠模式。
- 📈 **折线图** - 支持曲线/折线切换、面积图、堆叠模式。
- 💾 **全自动持久化** - 实时同步数据到 LocalStorage，刷新不丢失。
- 📥 **数据交互** - 支持 JSON/CSV 导入，支持跨设备存档导出。
- 🖼️ **高清导出** - 支持 PNG、JPG（适配主题）及 SVG 格式。
- 🌓 **全系统主题** - 深度适配深色/浅色模式。

## 🚀 快速启动

在终端中执行以下脚本即可快速开启或关闭服务：

### 1. 开启服务

```bash
./scripts/start.sh
```

*该脚本会自动启动服务器并为您打开浏览器。*

### 2. 关闭服务

```bash
./scripts/stop.sh
```

## 🛠️ 项目结构

```bash
chart-studio/
├── scripts/            # 启动与停止脚本
├── js/
│   ├── charts/         # 各图表逻辑 (pie.js, bar.js, line.js)
│   ├── data/           # 数据处理 (importer.js)
│   └── export/         # 导出逻辑 (exporter.js)
├── css/                # 视觉设计系统
└── index.html          # 主入口文件
```
