# Google Antigravity

Google Antigravity 是一个智能体开发平台，正将 IDE 进化到智能体优先的时代。Antigravity 让开发人员能够在更高维的任务导向层面上跨工作空间管理智能体，同时在核心保留了熟悉的 AI IDE 体验。

## 主要特性

* **AI 驱动的 IDE**：内置智能体 (Agent)、Tab 和 Command 等特性的 AI 驱动 IDE。
* **异步智能体**：可以在所有工作空间中并行工作的本地智能体。
* **智能体管理器 (Agent Manager)**：围绕规划模式、对话 UI 和产物审查构建的新视图。
* **多窗口**：包含编辑器 (Editor)、管理器 (Manager) 和浏览器 (Browser) 的产品。
* **浏览器子代理 (Browser Subagent)**：可以驱动浏览器执行 UI 测试和 SCM 操作等开发任务的智能体。

## 核心界面

* **编辑器 (Editor)**：功能齐全的 AI 驱动 IDE。
* **浏览器 (Browser)**：（预览版）能够读取和操作 IDE 之外界面的智能体能力。
* **智能体管理器 (Agent Manager)**：（预览版）用于任务、对话和产物的编排视图。
