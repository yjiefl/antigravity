# Antigravity 编辑器：命令 (Command)

命令功能将自然语言的力量直接带入您的工作流中，允许您随时请求特定的行内补全或终端命令。

## 工作原理

* **触发**：按下 `Command + I` (Mac) 或 `Ctrl + I` (Windows/Linux)。
* **提示框**：在当前光标位置会出现一个文本输入框。
* **指令**：以自然语言输入您的请求（例如：“创建一个对该列表排序的函数”或“为此代码块添加错误处理”）。
* **执行**：Antigravity 会直接在行内生成代码或命令，供您审查和接受。

## 使用场景

### 在编辑器中

使用命令生成模板代码、重构复杂函数或编写文档，而不会中断您的编码流程。

* **示例**：“为登录表单创建一个 React 组件。”

### 在终端中

在集成的 Antigravity 终端中使用命令生成复杂的 Shell 命令，无需记忆语法。

* **示例**：“查找所有在 3000 端口监听的进程并杀死它们。”
