# 智能体管理器 (Agent Manager)

我们构建了智能体管理器，以提供在您的指导下 Antigravity 智能体所做工作的更高层级视图。在这里，您可以跨多个工作空间工作，同时监管数十个智能体，并主要通过智能体（而不是直接编写代码）与您的代码库进行交互。随着智能体和模型的不断完善，我们相信这种鸟瞰视图将成为您所有工作的主要入口。目前，随着我们不断收集反馈并迭代这一新界面，智能体管理器处于公开预览阶段。我们预计很快它将成为 Antigravity 体验的核心。

在任何时候，您都可以通过按下 `CMD+E` (Mac) 或 `CTRL+E` (Windows)，或者通过菜单栏右上角的 "Open Editor" 和 "Open Agent Manager" 按钮在智能体管理器和编辑器之间切换。您还可以通过管理器管理您的编辑器窗口，包括隐藏、聚焦或关闭它们。
