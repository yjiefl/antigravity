# 浏览器子代理视图 (Browser Subagent View)

## 概述

管理器有一个专用的侧边面板，允许您展开并检查智能体针对某项任务的工作。在常规管理器对话视图中，浏览器子代理的工作通常是隐藏的。点击展开按钮将调出子代理视图。智能体工作的更新将流式传输到此视图中，因此您可以关注并根据需要与步骤进行交互（例如：确认/拒绝操作）。

## 侧面板内容

* 所有子代理操作（点击、滚动、导航等）
* 显示点击确切位置的视觉反馈
* 每一个步骤捕获的屏幕截图

## 视觉检查功能

在浏览器中产生操作（如点击）的工具调用包含一个按钮，该按钮可以打开浏览器在那一时刻的屏幕截图，并用红点显示智能体在浏览器中执行的交互。
