# 知识 (Knowledge)

知识项 (Knowledge Items, KIs) 是 Antigravity 的持久记忆系统，它会自动捕获并组织您编码会话中的重要见解、模式和解决方案。它们帮助您在不同的对话中建立在之前工作的基础之上。

## 什么是知识项？

知识项是关于特定主题的相关信息的集合。每个知识项都包含一个描述其内容的标题和摘要，以及一个提供该主题信息的产物集合。产物的示例包括自动生成的文档、代码示例或用户指令的持久记忆。

## 知识项是如何生成的？

当您与智能体交互时，Antigravity 会自动分析并从您的对话中提取信息，并使用该信息创建新的知识项或更新现有的知识项。

## 查看知识项

您可以在 Antigravity 智能体管理器中查看您的知识项。

## 智能体如何使用知识项？

您所有知识项的摘要对智能体都是可见的，它使用这些摘要来优化其响应。当智能体识别出与对话相关的知识项时，它会自动研究该知识项中的产物并使用适用的信息。
