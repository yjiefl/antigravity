# 入门指南 (Getting Started)

## 下载
请访问 [antigravity.google/download](https://antigravity.google/download) 下载 Google Antigravity。
* **macOS**: 版本 12 (Monterey) 或更高。不支持 X86。
* **Windows**: Windows 10 (64 位)。
* **Linux**: glibc >= 2.28, glibcxx >= 3.4.25 (例如：Ubuntu 20, Debian 10)。

## 基础导航
* **智能体管理器 (Agent Manager)** 可以通过编辑器顶栏按钮或 `Cmd + E` 打开。
* **编辑器 (Editor)** 可以通过智能体管理器中的 "Focus Editor" 选项或 `Cmd + E` 打开。
