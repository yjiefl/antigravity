# 实现计划 (Implementation Plan)

智能体利用实现计划产物来架构代码库中的更改以完成任务。这些计划包含有关必要修订的技术细节，旨在由用户审查。

除非您的产物审查策略设置为“始终继续”，否则智能体通常会在进行完成任务所需的更改之前请求您审查实现计划。您可以点击对话中或产物标题处的“Proceed”按钮来立即继续执行智能体的计划。

通常，智能体创建的计划可能与您的确切需求略有不同。Antigravity 支持对这些产物进行评论，因此您可以出于任何原因向智能体提供反馈，无论是缩小更改范围、使用不同的技术栈，还是纠正智能体的不一致之处。

在实现计划上留下评论后，您仍然可以使用“Proceed”继续执行计划；或者，您也可以切换产物标题中的“Review”按钮，在那里您可以检查所有评论并留下消息作为反馈，而不是直接继续。

一旦您继续执行或留下了审查意见，智能体将继续其工作，要么迭代实现计划并重新请求您的审查，要么开始其实际工作！
