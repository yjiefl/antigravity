# 智能体模式 / 设置 (Agent Modes / Settings)

## 对话级模式
* **规划 (Planning)**：智能体在执行前进行规划；用于复杂的调研和协作工作。
* **快速 (Fast)**：智能体直接执行任务；用于重命名变量等简单的局部任务。

## 全局设置
“智能体”标签页中的主要设置包括：
* **产物审查策略 (Artifact Review Policy)**：选项包括“始终继续”或“请求审查”（停止以等待用户批准）。
* **终端命令自动执行 (Terminal Command Auto Execution)**：可配置的自动执行 Bash 或 PowerShell 命令的“允许”和“拒绝”列表。
* **智能体非工作空间文件访问**：控制对当前工作空间以外文件的访问（出于安全考虑，默认禁用）。
