# 模型 (Models)

## 推理模型
Antigravity 提供来自 Google Vertex Model Garden 的模型，包括：
* Gemini 3 Pro (high/low) 及 Gemini 3 Flash
* Claude Sonnet 4.5 及 Claude Opus 4.5
* GPT-OSS

## 其他模型
* **Nano Banana Pro**：用于生成型图像任务（模型图、图表）。
* **Gemini 2.5 Pro UI Checkpoint**：由浏览器子代理用于操作执行。
* **Gemini 2.5 Flash**：用于后台检查点记录和摘要。
* **Gemini 2.5 Flash Lite**：用于代码库语义搜索。
