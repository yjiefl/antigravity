# 浏览器 (Browser)
Antigravity 具有打开、读取和控制 Chrome 浏览器的能力，允许您测试开发中的网站、读取互联网数据源以及自动化各种不同的浏览器任务。

通过使用[子代理](/docs/browser-subagent)，Antigravity 可以在浏览器上根据需要进行操作，并[记录其操作](/docs/browser-recordings)，并呈现选定的屏幕截图和视频作为[产物](/docs/screenshots)。

为了将 Antigravity 智能体与您的正常浏览隔离，它运行在一个单独的浏览器配置文件中。这意味着它在您的停靠栏（Dock）中将显示为一个单独的应用程序，并且不会登录任何账号。您可以在[独立的 Chrome 配置文件](/docs/separate-chrome-profile)部分阅读更多相关信息。

要禁用所有浏览器工具的使用，您可以在设置的“浏览器”部分禁用浏览器工具设置。

Antigravity 会检测并使用您现有的 Chrome 应用程序。如果您还没有 Chrome，必须在此处下载。如果 Antigravity 无法检测到您的 Chrome 安装路径，您可能需要在以下设置中手动指定路径。
