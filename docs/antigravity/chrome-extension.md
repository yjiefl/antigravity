# Chrome 扩展程序 (Chrome Extension)

Antigravity 浏览器扩展程序是 Antigravity 智能体访问 Web 的必要条件。它使智能体能够查看网站并与其交互，以完成您的开发目标，无论您是从头开始构建网站还是自动化工作流。

它通过允许用户从浏览器取消当前对话、将焦点从 Web 智能体正在工作的页面切回 Antigravity 等操作，增强了用户体验，并使与浏览器智能体的并行工作更加无缝。

第一次使用 Antigravity 浏览器智能体时，您应该会被引导至 Chrome 网上应用店安装该扩展程序。

## 故障排除

如果出于任何原因需要手动安装：

1. 点击 Antigravity 中的 Chrome 图标（使用 Antigravity 用户配置文件打开 Chrome）
   * 在编辑器中，您可以在右上角找到 Chrome 图标
   * 在智能体管理器中，您可以在左下角找到 Chrome 图标
2. 导航至该 URL 并点击“添加至 Chrome”
