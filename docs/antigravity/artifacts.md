# 产物 (Artifacts)

我们将“产物”定义为智能体为了完成工作或向人类用户交流其工作和思考而创建的任何东西。这些包括丰富的 Markdown 文件、Diff 视图、架构图、图像、浏览器录制、代码 Diff 等。随着智能体变得越来越自主，并能运行越来越长的时间，产物允许智能体异步地向用户传达其工作，而不是要求用户同步地仔细监视智能体的每一步。

产物是在智能体处于“规划模式”时生成的，并同时出现在智能体管理器和编辑器视图中，尽管前者针对显示、组织和管理产物进行了优化。

反馈是产物的另一个关键概念。根据用户设置，智能体可能会要求审查中间产物，以确认其思维或实现中的进展符合用户的意图和目标。用户可以对产物提供反馈，以引导智能体朝着正确的方向发展。不同类型的产物，反馈的 UI/UX 也有所不同。
