# 附录 E 作业风险评估规范
> （规范性附录）

## 1. 定义

**1.1 危害**：可能导致伤害或疾病、财产损失、工作环境破坏或这些情况组合的条件或行为。

**1.2 风险**：某一特定危害可能造成损失或损害的潜在性变成现实的机会，通常表现为某一特定危险情况发生的可能性和后果的组合。

**1.3 风险评估**：辨识危害引发特定事件的可能性、暴露和结果的严重度，并将现有风险水平与规定的标准、目标风险水平进行比较，确定风险是否可以容忍的全过程。

## 2. 要求与方法

### 2.1 区域内部风险评估

主要针对作业任务执行过程进行，旨在掌握危害因素分布及风险大小。评估结果需填写《区域内部风险评估填报表》。

#### 2.1.1 填报要素说明

- **工种**：电力生产活动中专业作业活动的分类（如调度、输电、变电、配电等）。
- **作业任务**：各专业涉及的工作任务类别，同类项需合并。
- **作业步骤**：按执行功能分解的阶段，应避免划分过细，按功能单元划分。
- **危害名称**：步骤中可能危及人员、设备、电网和形象的称谓。未涉及的内容格式为“副词+名词/动名词”。
- **危害类别**：分为物理、化学、机械、生物、人机工效、社会-心理、行为、环境、能源 9 大类。
- **风险后果**：包括人身伤残/死亡、设备损坏、事故/事件、健康受损、环境污染/破坏。

#### 2.1.2 风险种类与范畴对照表

| 风险范畴 | 细分风险种类 |
| :--- | :--- |
| 安全-人身 | 坠落、灼（烫）伤、摔绊、扭伤、坍塌、触电、交通意外、夹伤、碰撞、打击、剪切、割伤、刺伤、绞伤、中毒、窒息、咬伤、淹溺、感染、爆炸等 |
| 安全-设备 | 设备烧损、设备疲劳损坏、设备性能下降、设备破损、设备报废、设备停运等 |
| 安全-电网 | 电压波动、频率波动、系统振荡、系统瓦解、局部停电、大面积停电等 |
| 健康 | 职业病、职业性疾病（听力/视力受损、肺功能障碍、腰肌劳损等）、心理伤害、精神障碍 |
| 环境 | 土壤污染、水污染、大气污染、生态失衡、工作环境污染等 |
| 社会责任 | 企业声誉形象受损、供电中断、客户投诉 |

#### 2.1.11 风险等级分析

**评估公式：**

$$ R = S \times E \times P $$
*(风险值 = 后果 × 暴露 × 可能性)*

**(1) 后果 (S) 分值表**

| 序号 | 安全后果 | 分值 |
| :---: | :--- | :---: |
| a | 1人以上死亡或重伤；损大于100万元；一般以上电力事故 | 100 |
| b | 2人以上轻伤；损失30万-100万元；三级以上电力事件 | 50 |
| c | 1人轻伤；损失20万-30万元；四级电力事件 | 25 |
| d | 1人以上轻微伤；损失10万-20万元；五级电力事件 | 15 |
| e | 损失1万-10万元；六级电力事件 | 5 |
| f | 损失1000元-1万元；七、八级电力事件 | 1 |

**(2) 暴露 (E) 分值表**

| 序号 | 引发事故序列的第一个意外事件发生的频率 | 分值 |
| :---: | :--- | :---: |
| a | 持续（每天许多次） | 10 |
| b | 经常（大概每天一次） | 6 |
| c | 有时（每周一次到每月一次） | 3 |
| d | 偶尔（每月一次到每年一次） | 2 |
| e | 很少（据说曾经发生过） | 1 |
| f | 特别少（没有发生过，但有发生的可能性） | 0.5 |

**(3) 可能性 (P) 分值表**

| 序号 | 事故序列发生的可能性（安全、环境、社会责任） | 职业健康可能性 | 分值 |
| :---: | :--- | :--- | :---: |
| a | 100% 产生预期结果 | 频繁（每6个月一次） | 10 |
| b | 十分可能（50%） | 持续（每1年一次） | 6 |
| c | 可能（25%） | 经常（每1-2年一次） | 3 |
| d | 很少的可能性 | 偶然（3-9年一次） | 1 |
| e | 相当少但确有可能 | 很难（10-20年一次） | 0.5 |
| f | 百万分之一的可能性 | 罕见（几乎从未发生过） | 0.1 |

#### 2.1.11.4 风险等级划分与对策

| 风险等级 | 风险值范围 | 应对措施 |
| :--- | :--- | :--- |
| 特高风险 | ≥ 400 | 考虑放弃、停止 |
| 高风险 | 200 ≤ 风险值 < 400 | 需要立即采取纠正措施 |
| 中等风险 | 70 ≤ 风险值 < 200 | 需要采取措施进行纠正 |
| 低风险 | 20 ≤ 风险值 < 70 | 需要进行关注 |
| 可接受风险 | < 20 | 容忍 |

#### 2.1.15 控制措施判断（J）

对风险值 > 70 的评估结果，需提出控制措施建议。

**判断标准**：

* **J ≥ 10**：预期的控制措施费用支出恰当。
* **J < 10**：预期的控制措施费用支出不恰当。

### 2.2 区域外部风险评估

针对作业活动区域外的危害因素（自然灾害、地理环境、外界人员/物质）进行评估，目的是为应急管理提供输入。其余填报要求与区域内评估相同。
