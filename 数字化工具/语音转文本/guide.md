# 使用指南 (User Guide)

本指南介绍如何配置和使用语音转字幕工具。

## 1. 配置 API Key

如果你打算使用 OpenAI 或 Groq 引擎，请按照以下步骤操作：

1.  进入 `server` 目录。
2.  打开 `.env` 文件。
3.  填入相应的 Key：
    ```env
    OPENAI_API_KEY=sk-xxxx
    GROQ_API_KEY=gsk-xxxx
    ```
    _注：你也可以直接在网页界面上输入 Key。_

## 2. 启动服务

在项目根目录下，直接运行：

```bash
./start.sh
```

该脚本会执行以下操作：

- 检测并关闭之前运行的服务。
- 启动 Python 后端（FastAPI）。
- 启动前端项目（Vite）。
- 自动在浏览器中打开 UI。

## 3. 选择转写模式

- **OpenAI Whisper**: 官方 API，效果最稳，按时长计费。
- **Groq**: 速度最快，适合追求效率的用户，目前有免费公测额度。
- **本地 Whisper**: 推荐隐私要求高或网络不佳时使用。首次加载较慢，之后速度取决于本机性能。

## 4. 处理超大视频

如果你的视频非常大（超过几百 MB），请给后端一些处理时间。脚本会自动进行如下处理：

1.  **压缩**: 将音频提取并转换为低位率 MP3。
2.  **切片**: 如果压缩后仍超过 API 限制，会切分为 10 分钟一段。
3.  **合并**: 自动整理时间轴并合并为完整的 SRT 文件。

## 5. 停止服务

当你不再使用时，运行：

```bash
./stop.sh
```

这会释放系统端口并停止所有相关后台进程。
