# 运维合同结算系统 Demo 使用说明

## 1. 系统简介

本系统是一个用于演示运维合同季度结算流程的单页面 Web 应用 (SPA)。它模拟了甲方管理员进行费用核对、调整和汇总的真实场景。

## 2. 核心功能

- **标段库管理**：系统内置了合同标段与价格模板的对应关系。
- **季度结算自动化**：针对“包干费用”类项目，系统自动按季度比例（25%）计算当前理论结算额。
- **费用弹性调整**：支持对任何分项进行金额调增或调减。
- **强制性原因说明**：根据审计要求，所有非理论数值的调整必须配套文字说明。
- **实时汇总**：所有数值变动即时反馈在页面顶部的统计卡片和底部的总计行中。

## 3. 操作指引

1. **进入结算界面**：点击左侧导航栏的“季度结算申报”。
2. **选择对象**：在上方下拉框中选择需要申报的“场站”和“所属季度”。
3. **填写数据**：
   - 对于非包干项目（如人工费），在“本次结算数量”中输入实际数量。
   - 系统将根据单价实时计算“理论结算额”。
4. **进行调整**：
   - 如果需要手动干预金额，在“费用调整额”列输入数值（可正可负）。
   - 输入后，点击最右侧的“调整备注”单元格，在弹出的窗口中输入调整依据。
5. **提交申报**：核对无误后，点击右下角的“确认并提交单据”。

## 4. 技术规格

- **文件**：`index.html` (包含所有 HTML, CSS 和 JS)
- **依赖**：无需后端服务器，本地双击即可运行。
- **浏览器兼容性**：支持 Chrome, Edge, Safari 等现代浏览器。
