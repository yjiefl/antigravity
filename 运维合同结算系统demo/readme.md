# 运维合同结算系统demo

## 需求

制作demo，用于真实开发参考。
根据运维合同价格表为基础，制作结算系统，由乙方填写各分项数量，并提供支撑材料，由甲方确认后，系统自动生成结算单。

## 技术栈

用html单文本软件实现。

## 功能

1. 制作标段库，场站根据不同标段对应不同标段的价格
2. 包干费用根据季度结算，每季度结算每季度的钱
3. 对于结算费用可以根据实际情况修改，说明原因
4. 最终形成结算汇总表，汇总和查阅
