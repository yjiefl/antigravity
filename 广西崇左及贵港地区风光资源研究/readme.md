# 广西崇左及贵港地区风光资源研究 - 文档转换

本目录包含了将该研究课题相关的原始 Word 文档转换为 Markdown 格式的工具及结果。

## 目录结构

- `docs/`: 原始 .docx 文档。
- `markdown/`: 转换后的 .md 文档及提取的媒体文件。
- `log/`: 转换过程日志。
- `convert_docs.py`: 自动化转换脚本。
- `spec.md`: 项目需求规格说明。
- `guide.md`: 使用说明。
- `update.md`: 任务记录。

## 快速开始

转换结果已生成，可直接在 `markdown/` 目录下查看。
若需重新批量转换，请运行 `python3 convert_docs.py`。
