# 使用指南 (Guide)

## 1. 环境准备

- **基础环境**: 任何支持 HTML5 的现代浏览器 (推荐 Chrome, Safari, Edge)。
- **网络连接**: 初次运行需要访问网络以加载 CDN 资源 (Tailwind CSS, Chart.js)。

## 2. 运行项目

### 2.1 桌面版

1. 进入项目文件夹。
2. 双击 `index.html` 或将其拖入浏览器中。
3. 无需安装 Python、Node.js 或任何后端服务。

### 2.2 移动端版 (React)

1. 进入 `mobile` 文件夹。
2. 运行 `npm install` (初次运行)。
3. 运行 `npm run dev` 启动开发服务器。
4. 使用手机浏览器访问显示的局域网地址。

## 3. 核心算法说明

### 光伏功率计算

1. **理论直流功率 (DC)**: `DC = Capacity_DC * (Irradiance / 1000)`
2. **潜在交流功率 (AC_pot)**: `AC_pot = DC * PR` (PR 系统效率，本项目默认取 0.85)
3. **实际交流输出 (AC_act)**: `AC_act = min(AC_pot, Capacity_AC)` (受逆变器额定容量 100MW 限制)

## 4. 常见问题

- **页面显示空白**: 请检查网络连接，确保能够加载 CDN 资源。
- **滑块响应缓慢**: 建议在高性能浏览器上运行，图表渲染需要一定的计算资源。
