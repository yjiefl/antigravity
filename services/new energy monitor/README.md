# New Energy Monitor (新能源监控平台)

这是一个基于 **单文件 HTML** 架构的新能源监控系统演示项目，具有高度的可移植性和极简的部署流程。

## 🌟 特色功能

- **单文件部署**: 所有的样式 (Tailwind CSS)、图表 (Chart.js) 和逻辑均集成在单个 HTML 文件中。
- **实时模拟与分析**: 动态计算光伏电站在不同辐照度下的交流输出，支持截峰 (Clipping) 监测。
- **现代化 UI**: 采用响应式设计，适配桌面与移动端，提供高端的视觉体验。
- **移动端应用**: 提供专为手机优化的 React 移动端版本，支持 PWA 安装。
- **快速交互**: 通过滑块实时调整辐照参数，瞬时查看计算结果。

## 🛠️ 技术栈

- **前端核心**: HTML5, Vanilla JavaScript (ES6+)
- **样式方案**: Tailwind CSS (CDN 加载)
- **图表展示**: Chart.js
- **字体服务**: Google Fonts (Inter)

## 🚀 快速开始

只需在浏览器中直接打开 `index.html` 即可运行，无需搭建后端环境或安装依赖。
