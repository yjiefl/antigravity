# 数据曲线分析系统 (DataCurve Analysis System)

一个优雅、高效的 Web 数据分析工具，专注于时间序列数据的曲线展示与对比分析。

## 主要功能

- **指标聚合**: 自动计算 AUC (面积/积分) 及均值，支持风速、功率、辐照度等业务逻辑。
- **历史数据集成**: 集成 Open-Meteo 气象 API，支持按场站名自动匹配地理坐标并获取 15 分钟分辨率的历史辐照度数据。
- **持久化与日志**: 基于 SQLite 的快照管理，完善的访问及错误日志记录。
- **多维对比**: 轻松对比同一天内不同批次或不同维度的数据。
- **业务适配**: 自动识别电力行业核心指标（AGC、超短期）并预设标准限值。
- **数据管理**: 支持历史快照存单，数据库持久化存储，不因系统更新而丢失。
- **极致美学**: 现代感十足的磨砂玻璃风格 UI，支持深浅主题切换。

## 开发环境

- **操作系统**: macOS
- **运行环境**: Node.js 18+
- **构建工具**: Vite

## 快速开始

```bash
cd frontend
npm install
npm run dev
```

## 部署说明

### 1. 本地 NAS 部署 (QNAP)

确保本地网络通畅并在项目根执行：

```bash

./deploy_to_nas.sh
```

### 2. VPS 部署 (Docker)

确保 VPS ($IP: 107.174.62.30) 运行环境已安装 Docker 及其 Compose 插件：

```bash

./deploy_to_vps.sh
```
