
# 岑凡故障组串数据处理

本项目用于自动化处理 `岑凡现场提供故障组串数据表.xlsx`，将其扁平化并识别特定颜色标记的状态。

## 主要功能

- 解析 Excel 单元格背景色（红/黄）。
- 识别空支路。
- 提取箱变与逆变器号。
- 生成标准化的故障清单。

## 使用说明

1. 确保安装了 `openpyxl` 库：

   ```bash
   pip install openpyxl
   ```

2. 运行处理脚本：

   ```bash
   python process_xlsx.py
   ```

## 字段说明

- **集电线路**: 所在的集电线路名称。
- **箱变号**: 对应的箱变编号。
- **逆变器号**: 对应的逆变器编号。
- **组串号**: 1-24 号组串。
- **组串状态**:
  - **空支路**: 单元格无内容。
  - **风灾未修复**: 红色背景。
  - **零支路**: 黄色背景。
