# 现场安全风险管控系统

## 项目背景
为加强现场二次作业安全管理，本系统通过结构化的表格模板，将“作业任务”、“作业步骤”与“风险评估”进行深度关联，旨在提高风险辨识效率，规范作业方案编写。

## 主要功能
1.  **标准化流程**: 将典型作业任务拆解为标准步骤。
2.  **自动风险计算**: 依据《作业风险评估规范》实现风险值的自动计算与分级。
3.  **联动查询**: 选择任务后自动带出对应的危害预控措施，减少重复录入。

## 文件组成
*   `spec.md`: 项目设计规范文档。
*   `作业风险管控系统.xlsx`: 系统核心表格（待生成）。
*   `参考资料/`: 包含原始规范、计划及清单。

## 使用说明
1.  维护“标准风险库”与“方案模板库”。
2.  在“风险管控表”中通过下拉框选择作业任务。
3.  确认生成的风险信息，并根据现场实际情况微调 S/E/P 分值。
