# 项目快速使用指南

## 仓库结构概览

本仓库已按照功能模块进行了重新分类：

- **AI**: 存放 AI 技能插件、提示词工程等。
- **Apps**: 存放完整的、可运行的桌面或 Web 应用程序。
- **Services**: 存放各种中间件、数据处理服务等。
- **Docs**: 查阅本地化的 Antigravity 开发文档。

## 常用操作

### 查阅文档

所有翻译后的 Antigravity 文档均位于 `docs/antigravity/` 目录下，您可以直接在编辑器中阅读 Markdown 文件。

### 公文范本

位于 `docs/公文范本/` 目录下，包含经过格式清理和脱敏处理的公文模板 (`公文范本_cleaned.md`) 及相关处理脚本。

### Git 同步操作

本项目提供了自动化同步脚本 `git_sync.sh`。在开始工作或完成阶段性任务后，可以运行该脚本进行一键同步：

```bash
./git_sync.sh
```

该脚本将自动执行 `pull`、`add`、`commit` 和 `push` 操作。如遇冲突会停止并提示。

## 记录与跟踪

每次重要的操作及其结果都会记录在 `log/debug.log` 中。
更新记录和待办事项请参考各子项目下的 `update.md` 文件。

## 开发规范

请遵循 `docs/antigravity/spec.md` 中定义的 S.O.L.I.D 原则和文档组织规范。
