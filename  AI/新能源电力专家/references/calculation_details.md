# 新能源项目日报指标说明与计算手册 (2026版)

本手册详细说明了新能源项目（风、光、储）的核心指标定义、计算公式及分摊原则，作为审核《安全生产总调度会》材料的确定性计算依据。

## 1. 电量与损失指标

### 1.1 理论发电量

* **光伏**: 定义依据辐照度及直流侧容量。
* **风电**: 依据标准风功率曲线拟合，15分钟一个功率做积分。
  * 公式：$E = \sum P(v_i) \cdot F(v_i)$
  * $v_i$: 风速区间代表值；$P(v_i)$: 该风速下的功率曲线值；$F(v_i)$: 该区间持续小时数。

### 1.2 计划损失电量

* **统计范围**: 光伏场站停电时长统计范围为 **07:00 - 19:00**。
* **风电**: 停电时长结合风机测风仪风速及功率曲线计算。

### 1.3 故障损失电量

* **光伏故障损失**: `(故障时长[07-19点] / 12 * 等效利用小时数) * 故障容量`
* **风电故障损失**: `(故障时长 * 等效利用小时数) * 故障容量`
* **注意**: 光伏使用 **直流侧容量**，风电使用 **交流侧容量**。

### 1.4 限电率

* **公式**: `限电率 = 限电量 / (限电量 + 发电量) * 100%`
* **排除项**: 不包含分布式光伏项目。

## 2. 厂用电量计算与分摊原则

综合站用电量包含：主变耗电 + SVG耗电 + 集电线耗电 + 站用变电量 + 接地变耗电 + 汇流线耗电。

## 3. 储能系统特定指标

### 3.1 储能分摊逻辑

* **弄滩储能**: 储能站变耗电 + 40% * 弄滩#1主变 + 单元固定损耗。
* **樟木储能**: 储能站变耗电 + 20% *樟木#1主变 + 20%* 樟木#2主变 + 单元固定损耗。
* **守旗储能**: 储能站变耗电 + 50% * 守旗#2主变 + 单元固定损耗。

### 3.2 储能计算公式

* **充放电效率**: `(储能系统充电量 / 储能系统放电量) * 100%` (注意：原件中写为充电/放电，需按实际物理逻辑校验)。
* **固定损耗功率 (MW)**:
  * 守旗: 0.3363 MW
  * 弄滩: 0.3282 MW
  * 樟木: 0.0953 MW

## 5. 关键比例与魔法数字 (Magic Numbers)

### 5.1 守旗储能限电量分摊比例 (51.30%)

* **数值**: **51.30%**
* **应用逻辑**: 在计算守旗储能站承担的区域限电量时，`守旗储能限电量 = 中原站限电量 + 南部站限电量 * 51.30%`。
* **依据**: 该数字源于南部站（岜盆地块）接入守旗汇集站的协议容量比例。
  * `南部项目总容量 (162.75MW) = 岜盆 (83.5MW) + 岑凡/上屯 (79.25MW)`
  * `分摊占比 = 岜盆容量 (83.5) / 总容量 (162.75) ≈ 51.305%`。
  * 在审核时，须严格按照 51.30% 校验储能表的勾稽一致性。
