# 写作专家 (Writing Expert)

## 简介

这是一个专为中国公文写作设计的 AI 技能库，赋能 AI 助手成为资深的文秘专家。它集成了党政机关公文规范、行业专业术语以及智能化的写作流程。

## 目录结构

- `SKILL.md`: 核心指令与能力定义。
- `spec.md`: 技术方案与架构说明。
- `references/`: 知识库，包含各业务领域的参考手册。
- `scripts/`: 辅助脚本。
- `input/`: 输入素材存放处。
- `output/`: 成果输出存放处。
- `log/`: 调试日志。

## 核心功能

1. **公文起草与迭代**: 支持多轮反馈修改。
2. **文本审核与诊断**: 生成《公文质量诊断表》。
3. **亮点挖掘与提炼**: 从非结构化数据中提取价值。
4. **模板驱动写作**: (New) 依据特定模板进行精准创作。
5. **多源物料整合**: (New) 融合多个材料重构新文档。

## 使用说明

1. 将需要参考的素材或模板放入 `input/` 文件夹（可选）。
2. 在对话中指明任务类型（起草、审核、整合、模板写作）。
3. 按照 AI 提示补充必要的信息或确认大纲。
4. 在 `output/` 文件夹查看生成结果。
