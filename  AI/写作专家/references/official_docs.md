# 公文写作参考手册

本手册基于《公文范本_cleaned.md》提炼而成，旨在为“写作专家”提供标准化的公文写作规范、模板和常用语料。

## 一、 公文写作基本原则

1. **准确性**：内容真实，数据准确，表述严谨，无歧义。
2. **简洁性**：文字精炼，直奔主题，避免冗长空洞。
3. **规范性**：格式标准，文种使用得当，符合党政机关公文处理工作条例。
4. **政治性**：站位高，符合方针政策，体现组织意图。

## 二、 常用公文结构与模板

### 1. 通知 (Notice)

用于发布规章、转发上级公文、批转下级公文、要求下级机关办理和需要周知或共同执行的事项。

**结构：**

* **标题**：发文机关 + 事由 + 文种（如：xx公司关于xx的通知）
* **主送机关**：被通知的单位或部门
* **正文**：
  * **缘由**：依据、目的或背景（“为深入贯彻...”，“根据...要求”）
  * **事项**：具体要求、安排、措施（分条列出：一、...；二、...）
  * **结尾**：特此通知。
* **落款**：发文机关全称
* **日期**：xxxx年x月x日

**常用语：**

* 开端：现将有关事项通知如下：
* 结尾：特此通知。 / 请遵照执行。

### 2. 通报 (Circulated Notice)

用于表彰先进、批评错误、传达重要精神和告知重要情况。

**结构：**

* **标题**：关于xx的通报
* **主送机关**：所属各单位、各部门
* **正文**：
  * **情况概述**：事件经过、基本情况。
  * **分析/评价**：性质、意义、原因、教训。
  * **决定/要求**：表彰决定、处罚决定、后续要求（举一反三、引以为戒等）。
* **结尾**：特此通报。
* **落款** & **日期**

**常用语：**

* 表彰：成绩显著，涌现出一大批...，决定授予...称号。
* 批评：暴露了...问题，性质严重，影响恶劣。
* 要求：希望受表彰的...再接再厉；各单位要引以为戒...

### 3. 请示 (Request for Instructions)

适用于向上级机关请求指示、批准。

**结构：**

* **标题**：关于xx的请示
* **主送机关**：上级机关
* **正文**：
  * **请示缘由**：背景、依据、原因（“为...”，“由于...”）。
  * **请示事项**：具体请求批准的内容（方案、资金、项目等）。
  * **结尾**：妥否，请批示。 / 以上请示，妥否，请批复。
* **落款** & **日期**

**特点：** 一文一事，理由充分，要求明确。

### 4. 报告 (Report)

适用于向上级机关汇报工作、反映情况，回复上级机关的询问。

**结构：**

* **标题**：关于xx的报告
* **主送机关**：上级机关
* **正文**：
  * **导语**：简要说明报告的背景或依据。
  * **主体**：工作进展、成绩、存在问题、下一步打算（分条陈述）。
  * **结尾**：特此报告。 / 专此报告。
* **落款** & **日期**

**常用语：**

* 开端：现将有关情况报告如下：
* 汇报成绩：在...领导下，深入贯彻...，取得了...成效。

### 5. 函 (Letter)

适用于不相隶属机关之间商洽工作、询问和答复问题、请求批准和答复审批事项。

**结构：**

* **标题**：关于xx的函 / 关于xx的复函
* **主送机关**：相关单位
* **正文**：
  * **缘由**：说明发函的目的或背景。
  * **事项**：商洽的内容、询问的问题或答复的意见。
  * **结尾**：特此函达。 / 特此函告。 / 特此复函。
* **落款** & **日期**

### 6. 纪要 (Minutes)

适用于记载会议主要情况和议定事项。

**结构：**

* **标题**：xx会议纪要
* **导言**：时间、地点、主持人、参会人员、会议主题。
* **主体**：
  * 会议学习了...
  * 会议听取了...
  * 会议研究了... / 会议明确了...（列出决定事项、分工、要求）。
* **结尾**：（一般不需专门结尾，或写“纪要如下：”在导言后）

**常用语：**

* 会议指出...，会议强调...，会议要求...

### 7. 批复 (Reply)

适用于答复下级机关请示事项。

**结构：**

* **标题**：关于xx的批复
* **主送机关**：请示单位
* **正文**：
  * **引语**：你单位《关于xx的请示》（x号）收悉。
  * **批复意见**：经研究，批复如下：一、同意...；二、...。
  * **结尾**：此复。 / 特此批复。
* **落款** & **日期**

## 三、 排版与格式规范

1. **层级序号**：
    * 第一层：一、二、三、
    * 第二层：（一）（二）（三）
    * 第三层：1. 2. 3.
    * 第四层：（1）（2）（3）
2. **标点符号**：
    * 标题中一般不使用标点符号。
    * 结构层次序数，“一”后用顿号“、”，“（一）”后不用标点，“1”后用齐线黑点“.”或句点，“（1）”后不用标点。
3. **数字用法**：
    * 公文中的数字，除部分结构层次序数和在词、词组、惯用语、缩略语、具有修辞色彩语句中作为词素的数字必须使用汉字外，应当使用阿拉伯数字。
    * 成文日期：如“2021年4月30日”。
4. **Markdown 适配规范**：
    * **总标题**：使用 `# 标题内容`，确保文档结构清晰。
    * **逻辑层级与 Markdown 语法对应**：
        * 第一层（一、）对应 `## 一、`
        * 第二层（（一））对应 `### （一）`
        * 第三层（1.）对应 `1.` 或 **1.**
    * **数据突出**：关键KPI、电量或金额数字建议使用 `**数字**` 加粗强调。
    * **表格引导**：复杂数据对比应自动转化为 Markdown 表格，严禁使用纯空格对齐。

## 四、 常用词汇表

* **开端词**：为了、根据、按照、遵照、关于、兹、查、奉。
* **承转词**：为此、鉴于、结合、综上所述、总之。
* **结尾词**：特此通知、特此报告、特此函达、妥否，请批示、请遵照执行、请审阅。
* **表态词**：原则同意、准予、不予、照办。
* **称谓词**：我公司、你单位、贵局、该部门。

## 五、 范例片段

* **开头**：“......党组坚持以习近平新时代中国特色社会主义思想为指导，深入贯彻......，现将有关情况报告如下：”
* **强调**：“会议强调，要提高政治站位，增强‘四个意识’、坚定‘四个自信’、做到‘两个维护’......”
* **整改请求**：“针对......问题，我公司高度重视，立即......，现恳请......”
